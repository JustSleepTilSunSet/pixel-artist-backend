# pixel-artist-backend

## How to run?
  
  1. docker network create pixel-artist-dev-net
  2. docker-compose up -d --build
## debug:
  
  2. docker exec -it pixel-artist-backend bash
  3. flask run --host=0.0.0.0 --port=10000
